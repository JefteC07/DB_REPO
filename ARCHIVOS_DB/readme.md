# Documentación de la Base de Datos

Este repositorio contiene los archivos generados desde Navicat que describen y representan la base de datos utilizada en el proyecto. La información incluida permite comprender su estructura, relaciones internas y facilitar su recreación en cualquier entorno compatible.

Los archivos proporcionados son:

1. DB_Teamder_SQL.sql

Archivo exportado desde Navicat que contiene todas las sentencias SQL necesarias para crear la base de datos desde cero. Incluye:

Creación de tablas.

Definición de llaves primarias (PK) y llaves foráneas (FK).

Tipos de datos utilizados en cada columna.

Relaciones entre entidades.
Este archivo es útil para importar la base en otro gestor, revisar la arquitectura o realizar mantenimiento.

2. Modelo Relacional (Diagram-BaseTeamder.pdf)

Representación visual de la base de datos generada en Navicat. En este modelo se pueden observar:

Las entidades principales.

Sus atributos clave.

Las relaciones entre tablas y su cardinalidad.

Cómo fluye la información entre módulos del sistema.

Esta imagen permite comprender de manera rápida la lógica del diseño y sirve como referencia para desarrolladores, revisores o documentación académica.

En conjunto, estos archivos conforman la documentación básica necesaria para analizar, implementar o mejorar la base de datos dentro del proyecto.
